# Time of day

Create a time object

## Usage

``` r
time_of_day(hour = integer(), minute = integer(), second = numeric())
```

## Arguments

- hour:

  A numeric vector of hours

- minute:

  A numeric vector of minutes

- second:

  A numeric vector of seconds

## Value

A time_of_day vector object, stored as a vctrs record containing hours,
minutes and seconds.
