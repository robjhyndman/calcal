# calcal: Calendrical Calculations

An R implementation of the algorithms described in Reingold and
Dershowitz (4th ed., Cambridge University Press, 2018)
[doi:10.1017/9781107415058](https://doi.org/10.1017/9781107415058) ,
allowing conversion between many different calendar systems. Cultural
and religious holidays from several calendars can be calculated.
